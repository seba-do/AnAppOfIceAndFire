# A Compose-App of Ice and Fire

An App to show the houses from the [API of Ice And Fire](https://anapioficeandfire.com/).

![Overview](images/overview.png)
![House Details](images/house_details.png)

## Technologies

- Jetpack Compose
- Navigation-Compose
- Hilt
- Paging3 (RemoteMediator)
- Room
- Retrofit
- Moshi
- MVVM
- KotlinDSL
